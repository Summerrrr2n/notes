文件名大小写不敏感

本周遇到一个问题是一开始一个.ts文件的后缀不小心写成了.TS ,反反复复pull push了好几次，每次都要重新修改回.ts 。后来才知道只是更改文件名的大小写在github看起来是没有修改的，必须要先把文件名字改成别的，push之后再改回原来的才行。
